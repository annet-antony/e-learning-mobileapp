# E-Learning-Mobile-App
Learning Management System Front End

### Technology Used
1. React Native
2. React Native Paper
3. Expo

### Available Features
1. Student, Lecturer Create Account
2. Login
3. View Courses
4. Add Lesson Topics
5. View Lessons
6. Student Enrollment
7. Post Assigments

### Pre-requests
- Node.js installed
- Expo CLI installed
- Firebase project configured
- Expo CLI installed

### How to Run
1. Clone the repository
   git clone https://github.com/annet-antony/e-learning-mobileapp.git

2. Navigate to project directory
   cd e-learning-mobileapp

3. Install dependencies
   npm install

4. Start Expo server
   npx expo start

5. Run on:
   Android Emulator
   iOS Simulator
   Physical device using Expo Go app

6. Firebase Configuration
   Create a Firebase project and add your configuration inside:
   firebaseConfig.js
